Overview

The M.2 PN532 NFC Module is a compact and versatile NFC/RFID reader/writer module in the popular M.2 form factor. Built around the NXP PN532 chipset, it supports multiple contactless communication protocols and is perfect for payment systems, access control, and IoT applications.

Key Features

  • PN532 Chipset: Industry-standard NFC controller
  • Multiple Protocols: ISO14443A/B, FeliCa, MIFARE
  • M.2 Interface: Standard M.2 Key-E connector
  • Read/Write Support: Full NFC tag read and write capabilities
  • Flexible Integration: Works with various M.2 host boards
  • Open Source: Complete documentation and examples

Specifications

  • Chipset: NXP PN532
  • Interface: M.2 Key-E (2230)
  • Protocols: ISO/IEC 14443 Type A/B, FeliCa, MIFARE Classic/Ultralight
  • Operating Frequency: 13.56 MHz
  • Communication: I2C, SPI, UART
  • Operating Voltage: 3.3V
  • Operating Range: Up to 5 cm
  • Dimensions: M.2 2230 (22 x 30 mm)

Use Cases

  • Contactless payment terminals
  • Access control systems
  • Smart card readers
  • NFC tag programming
  • Product authentication
  • Data exchange applications

Getting Started

  1. Insert the module into an M.2 Key-E slot on your host board
  2. Install the required drivers and libraries
  3. Configure the communication interface (I2C/SPI/UART)
  4. Test with NFC tags or cards
  5. Integrate into your application

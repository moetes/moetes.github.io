Overview

Pitaya Go is a compact, feature-rich IoT development board powered by the Nordic nRF52840 SoC. It’s designed for battery-powered IoT applications with built-in battery management and charging capabilities.

Key Features

  • Powerful SoC: Nordic nRF52840 with Bluetooth 5
  • Battery Ready: Built-in LiPo battery charger and management
  • Rich Peripherals: Multiple sensors and interfaces
  • Low Power: Optimized for battery-powered applications
  • USB-C: Modern USB-C connector for programming and charging
  • Compact Size: Perfect for portable projects

Specifications

  • SoC: Nordic nRF52840
  • CPU: ARM Cortex-M4F @ 64 MHz
  • Flash: 1 MB
  • RAM: 256 KB
  • Wireless: Bluetooth 5.0, Thread, Zigbee
  • Battery: LiPo charger (up to 500mA)
  • USB: USB-C (USB 2.0)
  • Sensors: Temperature, accelerometer, ambient light
  • Dimensions: 35 x 45 mm

What’s Included

  • 1x Pitaya Go Board
  • 1x USB-C Cable
  • Documentation & Examples
  • Plastic enclosure (optional)

Perfect For

  • Wearable devices
  • Environmental monitoring
  • Smart home sensors
  • Asset tracking
  • Portable IoT projects

Getting Started

  1. Connect Pitaya Go to your computer via USB-C
  2. Install the development environment
  3. Try the pre-loaded examples
  4. Add a LiPo battery for portable operation
  5. Build your IoT application

Moetes Development Environment

January 16, 2026 by Moetes

Introduction

Setting up a development environment is crucial for efficiently creating and testing hardware products. At Moetes, we follow a structured approach to ensure our developers have the right tools and configurations to streamline the development process.

Choose the Right Platform

Selecting the appropriate platform is the first step. We typically use platforms like CircuitPython, Arduino, MicroPython, Raspberry Pi or custom development boards depending on the project requirements.

We will formally support the following platforms:

  • CircuitPython
  • Raspberry Pi
  • Arduino

And for the hardware platforms, we will verify our modules on the following development boards:

  • CircuitPython
    • Seeed Studio XIAO nRF52840
  • Raspberry Pi
    • Raspberry Pi 4 Model B
  • Arduino
    • Seeed Studio XIAO ESP32S3
    • Seeed Studio XIAO RP2040

Development Environment Setup

CircuitPython Setup

CircuitPython will be our primary development environment for most of our modules. Seeed Studio XIAO nRF52840 is our recommended development board for CircuitPython. We deeply resonate with the design philosophy behind the Seeed Studio XIAO series. The Moetes team shares this vision, prioritizing compact, efficient, and user-friendly designs in our own modules.

To set up CircuitPython on the Seeed Studio XIAO nRF52840, follow these steps:

  1. Bootloader (Optional): Update if necessary (Releases).
  2. Download: Get the latest CircuitPython UF2 from CircuitPython.org.
  3. Install: Connect board, double-click Reset to enter bootloader mode, then drag the UF2 file onto the mounted drive.
  4. Develop: Edit code.py on the CIRCUITPY drive to run your code.

Raspberry Pi Setup

Raspberry Pi Imager is the recommended tool for installing operating systems on Raspberry Pi devices. It simplifies the process of downloading and flashing OS images onto SD cards.

  1. Download Raspberry Pi Imager: Get it from the official Raspberry Pi website.
  2. Install the Imager: Follow the installation instructions for your operating system.
  3. Launch the Imager: Open the Raspberry Pi Imager application.
  4. Choose OS: Select the desired operating system for your Raspberry Pi (We recommend Raspberry Pi OS Lite).

Arduino Setup

We recommend using the Arduino IDE 2.x for a modern development experience. It offers an improved user interface, better code completion, and integrated debugging features compared to the classic Arduino IDE.

https://docs.arduino.cc/software/ide/#ide-v2
